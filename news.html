Состоялся релиз стабильной версии Microsoft Edge 130

19 октября 2024 22:01

Microsoft начала распространять стабильную версию Microsoft Edge 130 (130.0.2849.46), которая включает в себя несколько новых функций, улучшений и исправлений ошибок. В частности, разработчики улучшили раздел «Параметры» для упрощения поиска и навигации.

Исправления

  • Исправлена проблема, из-за которой не отображалась иконка портфеля на странице edge://settings/sidebar, если Edge Bar отключен с помощью политики StandaloneHubsSidebarEnabled.
  • Исправлена проблема, приводившая к некорректному отображению некоторых страниц SharePoint.

    • Анонсы

  • Живые субтитры. Политика LiveCaptionsAllowed появилась версии Microsoft Edge 103, но сама функция доступна не всем пользователям. До официального релиза её могут включить пользователи с политикой ExperimentationAndConfigurationServiceControl, установленной в значение 'FullMode'. О полноценном релизе компания сообщит дополнительно.
  • Диалоговое окно отмены для события beforeunload. Начиная с версии 131 меняется поведение диалогового окна отмены при событии beforeunload. Для предотвращения появлени диалога теперь нужно использовать event.returnValue = '' в обработчике события. Прежний метод с event.preventDefault работать не будет. Соответственно, политика BeforeunloadEventCancelByPreventDefaultEnabled теперь является устаревшей и не будет функционировать после версии 130.

    • Обновления компонентов

  • Привязка токенов. Функция Token Binding, использующая криптографические сертификаты для повышения безопасности токенов, является устаревшей и больше не поддерживается. Политика AllowTokenBindingForUrls также устарела.
  • Контекстные политики Copilot. Политика CopilotCDPPageContext заменена на EdgeEntraCopilotPageContext. Политика CopilotCDPPageContext признана устаревшей в версии 130 и будет полностью удалена в версии 133.
  • Улучшения раздела «Параметры». Чтобы сделать поиск настроек браузера более эффективным, разработчики внедряют три улучшения навигации. Функциональность всех параметров останется прежней, все настройки останутся на тех же страницах, что и сейчас.
  • Интеграция с Teams на macOS. Если Microsoft Edge выбран в качестве браузера по умолчанию для открытия ссылок в Teams, ссылки из чата Teams на macOS будут открываться в Edge, используя профиль, соответствующий профилю аутентификации пользователя в Teams. Это обеспечивает бесшовный переход между приложениями и синхронизацию пользовательских данных. Администраторы могут управлять этой функцией с помощью политики Microsoft 365 «Выберите браузер для открытия веб-ссылок».

    Новшества

    Обновление приложения «Ножницы»

    К существующим текстовым функциям, таким как копирование текста с изображения и редактирование конфиденциальной информации, добавлена опция «Копировать как таблицу». С её помощью можно легко скопировать структурированные табличные данные в электронное письмо, документ или таблицу. Для этого сделайте снимок экрана с таблицей, выберите «Действия с текстом» и затем «Копировать как таблицу». Теперь вы можете вставить в нужном месте структурированные табличные данные. Примечание: текущая версия функции лучше всего работает со снимками экрана, содержащими только одну таблицу
